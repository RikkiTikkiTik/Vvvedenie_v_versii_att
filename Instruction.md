# **Работа с Git и GitHub**
## 1. Проверка наличия установленного GIT
В терминале выполнить команду `git version`
Если Git установлен, то появится сообщение с информацией и версией программы, иначе будет сообщение об ошибке.
## 2. Установка Git
Загружаем последнюю версия Git с сайта [https://git-scm.com/downloads] Устанавливаем с настройками по умолчаниюS
